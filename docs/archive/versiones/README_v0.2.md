# TLS v0.2 - Base modular, SEO e i18n

## Objetivo

Consolidar la web TLS como landing corporativa modular, multilingüe y preparada para SEO técnico base.

## Elementos principales

- HTML5 + CSS3 modular.
- Bootstrap 5.
- Páginas: `index.html`, `services.html`, `contact.html`, `areas.html`.
- Carpetas de idioma: ES, EN, DE, FR, IT, PT, RU, PL.
- Sistema i18n mediante JSON.
- `robots.txt`.
- `sitemap.xml`.
- Schema LocalBusiness.
- Corrección de rutas de imágenes.
- Cards de servicios multidioma.
- Redirección desde botones de contacto hacia `contact.html#contact-form`.
