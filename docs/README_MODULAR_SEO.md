# LTS - Modularización controlada + SEO técnico base

## Fase 1: CSS modular
CSS separado en `base`, `header`, `hero`, `services`, `layout`, `footer` y `responsive`, importados desde `assets/css/main.css`.

## Fase 2: HTML semántico
Se conserva estructura con `header`, `section`, `article`, `nav`, `footer`, titulares y enlaces reales.

## Fase 3: Head SEO reutilizable
Incluye `title`, `description`, `canonical`, `hreflang`, Open Graph, favicon y `LocalBusiness`.

## Fase 4: Páginas base
Generadas: `index.html`, `services.html`, `contact.html`, `areas.html`.

## Fase 5: URLs por idioma
Generadas carpetas `/es/`, `/en/`, `/de/`, `/fr/`, `/it/`, `/pt/`, `/ru/`, `/pl/`.

## Fase 6: Sitemap, robots y schema
Incluidos `sitemap.xml`, `robots.txt` y schema `LocalBusiness`.

## Checklist
- responsive OK: probar móvil, tablet, escritorio.
- links OK: WhatsApp, email, Google Maps.
- idiomas OK: sustituir los JSON mínimos por traducciones completas.
- metadata OK: revisar dominio definitivo.
- velocidad OK: optimizar imágenes finales.
- SEO técnico OK: validar schema y sitemap.
